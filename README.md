# 🌐 Personal Portfolio Website

A simple, responsive, and modern personal portfolio website built using **HTML5, CSS3, Bootstrap 5, and JavaScript**. This project showcases my profile, education, hobbies, contact information, and an enquiry form in a clean and user-friendly interface.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* Fixed navigation bar with smooth scrolling
* Home section with profile and introduction
* Education section
* Hobbies section
* Contact section
* Enquiry form with JavaScript validation
* Bootstrap Icons
* Hover effects and smooth animations
* Simple and easy-to-understand code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Bootstrap Icons

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser, or use the Live Server extension for the best experience.
